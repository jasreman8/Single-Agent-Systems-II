# Single-Agent-Systems-II
This repo contains additional projects on Single Agent Systems.

Project 1: Automated Expense Claim Review System with Policy Enforcement and Financial Validation. This commit establishes the end-to-end agent that processes receipts, validates expenses, and automatically generates compliant summaries. 
- Document extraction (receipt text parsing, field extraction)
- Expense categorization using semantic classification logic
- Policy compliance checking against company rules
- Calculation and reporting module for totals, validations, and audit outputs
